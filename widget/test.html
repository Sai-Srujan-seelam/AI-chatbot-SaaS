WonderChat — Widget Test Page

This page simulates a client's website with the chat widget embedded.

Sample Business Content

Welcome to Oakton Family Dental. We provide comprehensive dental care for the whole family, including cleanings, fillings, crowns, implants, and cosmetic dentistry.

Our office hours are Monday-Friday 8am-5pm. We accept most major insurance plans. Call us at (555) 123-4567 to schedule your appointment.

How It Works

  1. Create a tenant via the Admin API
  2. Ingest the client's website content
  3. Embed the widget script tag on the client's site
  4. Visitors chat with an AI trained on the client's content

Embed Code

Replace YOUR_API_KEY with the key from tenant creation:

<script
    src="http://localhost:8000/static/widget.js"
    data-key="YOUR_API_KEY"
    data-api="http://localhost:8000"
    defer
></script>

Colors, position, icons, avatar, and branding are all configured server-side via the admin API. The widget fetches its config on load.

Public JS API

Control the widget programmatically after it loads:

WonderChat.open()        // open the chat window
WonderChat.close()       // close it
WonderChat.toggle()      // toggle open/close
WonderChat.sendMessage("Hi!")  // send a message
WonderChat.clearHistory() // clear chat history
WonderChat.isOpen()      // returns true/false
WonderChat.getSessionId() // returns session ID
